# "# Django-Mini-CMS" 
## How to run this project:
### DEVELOPMENT (WINDOWS):
##### Change to development settings:
    manage.py, wsgi.py, pytest.ini
##### run python: python manage.py runserver
### PRODUCTION:
##### Change to production settings:
    manage.py, wsgi.py, pytest.ini
##### see "Procfile"
##### upload to the server (Heroku)
